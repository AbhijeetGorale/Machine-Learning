## 📊 Student Analytics Dashboard

## 📌 Overview

This project demonstrates Data Visualization & Analytics using Python, Pandas, and Matplotlib.

`It performs` :

Exploratory Data Analysis (EDA)

Statistical Analysis

SQL-like operations with Pandas

Graphical Reporting & Dashboard Generation

The dataset (table.csv) contains student information such as:

Student ID, Name, Course, City, Gender

Marks, Attendance, Fees Paid, Batch

## ⚙️ Features

Load and inspect datasets with basic info (shape, columns, missing values).

Perform SQL-like analytics: grouping, aggregation, filtering.

Generate visualizations for insights:

Course-wise student count

Average marks by course

Gender distribution

City-wise student count

Attendance analysis

Fees collection by course

Marks distribution

## There are sample charts generated by the program in Data_Visualization_Analytics/Images


## 🚀 How to Run
Clone the repository or copy the script.

Place table.csv in the same directory.

## Run the program:

## bash

python Data_Analytics.py

## 🛠️ Tech Stack

`Python` 3.x

`Pandas` – Data manipulation & analysis

`Matplotlib` – Visualization
