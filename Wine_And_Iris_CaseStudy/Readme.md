## 📝 Case Studies Overview

### 🌸 Case Study 1: Iris Dataset (Decision Tree Classifier)
- **Objective:** Classify iris species based on sepal and petal measurements.
- **Dataset:** `iris.csv` (150 samples, 4 features, 3 species).
- **Techniques Used:** EDA, scatter plots, Decision Tree Classifier.
- **Outcome:** Achieved high accuracy (~95%) with clear decision boundaries.
- **Script:** `iris_decision_tree.py`

### 🍷 Case Study 2: Wine Dataset (KNN Classifier)
- **Objective:** Predict wine class using chemical composition features.
- **Dataset:** `WinePredictor.csv` (multi-feature dataset with class labels).
- **Techniques Used:** Data cleaning, scaling, hyperparameter tuning (K values).
- **Outcome:** Best accuracy achieved with K = 7 (~98%).
- **Script:** `wine_knn.py`
