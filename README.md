# 🚢 Titanic Survival Prediction using Logistic Regression

A Machine Learning project that predicts whether a passenger survived the Titanic disaster using Logistic Regression. The project demonstrates the complete Machine Learning workflow, including data loading, preprocessing, feature engineering, model training, and prediction.

---

## 📌 Project Overview

This project uses the famous Titanic dataset to build a binary classification model that predicts passenger survival.

The project covers:

- Loading the dataset
- Data cleaning and preprocessing
- Handling missing values
- Encoding categorical variables
- Feature selection
- Model training using Logistic Regression
- Model evaluation
